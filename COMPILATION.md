# InteractableSettings Compilation Instructions
## Requirements
- BONELAB_DIR environment variable must be set for assembly references - see [BoneLib/COMPILATION.md](https://github.com/yowchap/BoneLib/blob/main/COMPILATION.md)
- [BoneLib](https://github.com/yowchap/BoneLib) and [MelonLoader](https://melonwiki.xyz) must be installed to the PC version of the game (and the game ran at least once) for assembly references
- As of writing, Patch 6 of BONELAB is required. This mod may not build correctly on future versions of BONELAB, or the dependencies
